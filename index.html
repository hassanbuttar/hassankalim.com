Flagship Projects

Financial platforms designed for scale, resilience, and compliance

Payment Hub and Wallet

Cloud-native digital wallet and payment platform

Defined and governed architecture for a multi-channel, multi-currency wallet platform with modular microservices, transaction processing, reconciliation, and ledger management.

  • Kubernetes, Helm, infrastructure-as-code, and blue/green deployments.
  • RabbitMQ-based asynchronous communication for resilience and eventual consistency.
  • Ocelot API Gateway, OAuth2/JWT, rate limiting, service discovery, RBAC, TLS, and vulnerability scanning.
  • ELK, Prometheus, and Grafana observability supporting 99.95% uptime goals.

Mobile Banking Transformation

Microservices backend for real-time banking services

Re-architected monolithic banking APIs into domain-aligned services for user management, transactions, notifications, analytics, and partner integrations.

  • Isolated SQL Server databases per service with OpenAPI-based versioning.
  • Redis caching and RabbitMQ workflows to reduce API latency below 200ms.
  • Fault-tolerant Mastercard/SINNAD integrations using retry and fallback patterns.
  • Azure DevOps pipelines, OWASP scanning, JWT/RBAC, and mTLS-secured service communication.

International Remittance System

Mission-critical remittance and logistics architecture

Established architecture standards for a secure remittance ecosystem covering orchestration, partner integration, user access, auditability, and operational tooling.

  • On-prem Kubernetes cluster architecture with modular service scalability.
  • RabbitMQ-driven event design for domain decoupling and operational traceability.
  • JWT-based RBAC, centralized policy controls, and resilient Polly-based integration patterns.
  • Automated testing, deployment governance, and full-stack monitoring through Azure DevOps and Grafana.
